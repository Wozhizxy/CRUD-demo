一个基于Mongodb，art-template的前后端交互的小demo。

如果你想运行（大概率你不会想。）,尝试下面的命令：

npm install  

node app 

它会在你的计算机上的8080端口创建一个http服务，想查看效果，请在浏览器输入：

localhost:8080/list

（没有做根目录的路由，请注意访问/list路径。
